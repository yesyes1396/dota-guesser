Dotadle (24/7) — простой статический вариант игры Dotadle с русским интерфейсом.

Запуск (Python 3):

1. Откройте терминал в папке `dotadle-24-7`.
2. Запустите статический сервер:

```bash
python -m http.server 5000
```

3. Откройте в браузере: http://localhost:5000

Примечания:
- Для поиска вводите имена героев на английском (например: `Anti-Mage`, `Invoker`).
- Игра не ограничена по времени — нажмите "Новая игра" чтобы получить нового случайного героя.
